# 🐵 モンキーキャッチャー3D

サルゲッチュにインスパイアされた3Dアクションゲームです。逃げ回る猿たちを特殊なツールで捕まえよう！

## 🎮 ゲーム概要

天才科学者が発明した「知能増幅ヘルメット」を猿のリーダーが誤って装着してしまい、高い知能と悪の心を持ってしまいました。他の猿たちにもヘルメットをかぶせ、人類に反乱を起こそうとしています。プレイヤーは特殊な道具「キャプチャーネット」を使って逃げ回る猿たちを捕獲し、平和を取り戻す冒険に出ます。

## 🕹️ 操作方法

| キー | アクション |
|------|----------|
| W/A/S/D | 移動 |
| マウス | カメラ操作 |
| スペース | ジャンプ |
| Shift | ダッシュ |
| 左クリック | 攻撃/捕獲 |
| 1-5 | ツール切り替え |
| ESC | ポーズ |

## 🔧 ツール（ガチャメカ）

1. **🥅 キャプチャーネット** - 気絶した猿を捕獲
2. **⚡ スタンロッド** - 猿を気絶させる近接攻撃
3. **🚀 ブースター** - 高速ダッシュ移動
4. **🛸 ホバードローン** - 空中浮遊・滑空
5. **📡 レーダースキャナー** - 隠れた猿を探知

## 🐒 猿の種類

- **イエローモンキー** - 基本型、一直線に逃げる
- **ブルーモンキー** - 俊敏型、ジグザグに逃げる
- **レッドモンキー** - 攻撃型、反撃してくる
- **グリーンモンキー** - ステルス型、隠れている
- **ブラックモンキー** - エリート型、強敵

## 🗺️ ステージ

1. **ジャングルエリア** - 熱帯雨林（ボス: ジャングルキング）
2. **アイスマウンテン** - 雪山・氷河（ボス: フロストリーダー）
3. **遺跡ダンジョン** - 古代神殿（ボス: テンプルガーディアン）
4. **メカニカルシティ** - 未来都市（ボス: サイバーエイプ）
5. **ボルケーノベース** - 火山基地（ボス: メガスペクター）

## 🛠️ 技術スタック

- **フレームワーク**: React + TypeScript
- **3Dライブラリ**: Three.js (react-three-fiber)
- **状態管理**: Zustand
- **ビルドツール**: Vite
- **デプロイ**: GitHub Pages

## 🚀 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/Get-the-monkeys.git
cd Get-the-monkeys

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```
